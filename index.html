Introducción a Python

Phyton es un lenguaje interpretado, este es un lenguaje soporta la programación orientada a objetos. Tiene una licencia de codigo abierto y su sintaxis es más simplificada que otros lenguajes de programación. Fue creado por Guido van Rossum, el código se día a conocer en febrero de 1991.


El código creado contará con una gran legibilidad permitiendo una mejor comprensión por parte de los programadores y haciendo más fácil la implementación del mismo ya que podemos plasmar ideas complejas con un código muy reducido en comparación con otros lenguajes de programación. Cuenta con una gran comunidad y soporte.


Características

Python es capaza de forzar a los programadores de adquirir un estilo particular de programación porque es posible aplicar la programación funcional, imperativa y la orientada a objetos como se mencionó anteriormente, su lenguaje es dinamicamente tipado donde una variable puede tomar distintos valores.


El intérprete que tiene Python de forma estándar tiene un modo interactivo done se escriben las instrucciones, estas se pueden introducir una por una y veremos como resultado la evaluación inmediatamente, permite que probemos porciones de código en el modo interactivo antes de ser integrado a un programa.

Variables

Cada leguaje tiene su forma de implementar el concepto de variable, en varios lenguajes se entiende que es una caja la cual permite almacenar un valor representando una posición de memoria en el ordenador, pueden estar representadas mediante letras, números o palabras. La variable es creada cuando se le asigna un valor por primero vez y este valor es asignado mediante el operador de igualdad



          numero = 10
          nombre = "Omar Forero"
          decimal = 20.5

El comando print permite que interactuemos con el programa donde podemos observar los datos almacenados en las variables 



          palabra = "Soy un String"
          numero = 30

          # El siguiente comando muesta al usuario el texto
          print(palabra)

          # El siguiente comando muesta al usuario un número

Datos en Python

Este es un atributo que indica a la computadora y al programador que clase de datos se van a manejar, en Python tenemos estos principales tipos de datos:


  • Numeros complejos
  • Enteros
  • Cadenas
  • Boleanos
  • Flotantes

Enteros: son aquellos números positivos como negativos que no tienen decimales, estos se pueden representar mediante int, si es un número entero se realiza mediante long. Flotantes: son aquellos números positivos o negativos que tiene decimales y los podemos representar mediante float. Complejos: estos números tienen una parte real y una imaginaria se expresan mediante complex. Cadena: también se le conoce como string, este es un texto el cual se encuentra encerrado en comillas simples o dobles. Booleano: puede adquirir dos parámetros True o False son muy utilizados en las estructuras condicionales.
